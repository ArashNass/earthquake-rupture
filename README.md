# Earthquake Rupture Tool

**Live:** https://arashnassirpour.com/earthquake_rupture/

An interactive 3D block model for understanding earthquake fault geometry.
Pick a fault mechanism and adjust strike, dip, and rake to see exactly how the
two sides of a fault move relative to each other, with an animated slip motion.
Includes a side-by-side comparison of all six standard mechanism types.

Built as a single self-contained `index.html`. Deployed on Netlify; any commit
to `main` goes live automatically. Made for learning and exploration, not for
hazard assessment.

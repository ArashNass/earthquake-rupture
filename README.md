# Earthquake Rupture Tool

**Live tool:** https://arashnassirpour.com/earthquake-rupture/

An interactive 3D block model for understanding earthquake fault geometry and slip.

## Features

- Adjust strike, dip, and rake interactively.
- Animate relative movement across the fault plane.
- Explore normal, reverse, strike-slip, and oblique mechanisms.
- Compare all six standard mechanism types side by side.
- Use the visual relationship between fault orientation and slip direction as a teaching aid.

## Scope

The tool illustrates fault kinematics and terminology. It is not a rupture forecast, finite-fault inversion, ground-motion model, or site-specific hazard assessment.

## Run locally

The complete application is contained in `index.html`. Open it in a modern browser or serve the repository with any static web server.

## Deployment

GitHub Pages publishes `main` at the live URL above.
